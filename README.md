# blog-api

This repo contains a simple blog API to CRUD users, posts, likes and comments.
It is meant as a personal project to cover the following topics that I wanted to learn/practice:

- [x] commit hooks
- [x] linters / formatters
- [ ] validation
- [ ] DTOs
- [ ] authentication
- [ ] authorization / rbac
- [ ] repositories
- [ ] unit testing / TDD

Technologies used:

- TypeScript
- Express.js
- ESLint
- Prettier
- Husky
- Prisma ORM
- PostgreSQL
- Yarn
